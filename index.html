型号匹配回填工具

左表=要报的型号顺序 · 右表=能报的型号+数据 → 按型号匹配，回填到左表位置，一键复制回 Excel

① 左表：要报的型号清单（决定输出顺序）

每行一个型号。第一列当作型号；后面若有自带列也会保留。可直接从 Excel 复制粘贴。

② 右表：能报的型号 + 数据（数据源）

第一列=型号，后面所有列=要回填的数据（数量/品牌/单价/总价/库存/货期…你贴几列就回填几列）。
📷 用图片识别录入（可选 · 需 Gemini API Key）
上传截图/拍照，自动识别成「型号 + 数据」并填入上面对应的输入框。复用你项目里已在用的 Gemini（多模态，能直接读表格，无需另加 OCR）。 Key 只存在你本机浏览器（localStorage），不会上传到任何服务器（除 Google 官方接口）。
